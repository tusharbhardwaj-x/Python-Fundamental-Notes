# Python-Fundamental-Notes
A complete end-to-end collection of Python notes covering essential programming concepts. This repository includes Jupyter Notebooks on operators, data types, lists, sets, dictionaries, tuples, conditional statements, loops, functions, and modules. Designed for beginners and learners who want a structured path to mastering Python basics.

# Topics Overview

## Basics:
Variable: where we store multiple values
Numeric: int, float, complex
Sequence: string, list[], tuple()
Set: Set{}, Frozen Set{}
Mapping: Dictionary{} 
Boolean: True/False

## Operators:
Assignment Operator: =, +=, -=, *=, /=, **=, //=, %=
Airthmetic Operator: +, -, *, /, //, %
Comparison Operator: >, <, ==, !=, >=, <=
Logical Operator: and, or, not
Identity Operator: is, is not
Membership Operator: in, not in
Bitwise Operator: &, |, ~, ^, >>, <<

## Data Sets:
List[]: Mutable, Ordered, Duplicates allowed, important methods
Tuple(): Not Mutable, Ordered, Duplicates allowed, important methods, Conversion, Unpacking of tuples
Set{}: Not Mutable, Not ordered, No duplicates allowed, important methods, Joins(union, Intersection, Difference, Symmmetric Diffrence)
Dictionaries{Keys:Values}: Mutable, Ordered, No Duplicates, Accessing, NEsted Dictionaries

## Conditional Statement
User Inputs: Here we create a input section for users
If: holds true conditions only ex: value is divisible by 7 or not
If-else: Holds true and false conditions ex: Pass-Fail, even-odd, etc
elif: Holds multiple conditions ex:Grading system
Nested if: Using multiple ifs inside ifs ex: Login Page
Short Hand if: To use if in single line of code

## Loops
for loop: to ittrate the values present in data structure
          loop, end, break, continue, range, patterns
While loop: continue till a specific condition hits
            while, break, continue

## Functions:
def function: parameter and arguments
Arbitary functions: to call multiple argument in single function
lambda function: using functions in single line code
for loop in functions

## Modules:
Math: constants, ceil/floor, airthmetic function, trigbometric fuctions
Date & Time: Basics, strftime, alphabets
Random: random.randint, random.choice
Custome: To create our own module
